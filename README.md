
## Description
This app will get website links and check it in various forms:
- VirusTotal (API)
+ nslookup
+ whois
- SSL certificates
- ...

(App is not complete, and left for better times)
## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
